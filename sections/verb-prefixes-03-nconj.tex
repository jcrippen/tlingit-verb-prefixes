\section{\textit{N}-conjugation prefix \textit{n-}}\label{sec:nconj}

\begin{multicols}{2}
\noindent
The \fm{\Af{n}-} conjugation prefix appears in two separate classes of forms:
	(i) progressive aspect forms with \fm{\Af{n}-}
	and (ii) forms with a conjugation prefix \fm{\Af{\xx{cnj}}-} = \fm{\Af{n}-}.
The progressive aspect uses \fm{\Af{n}-} in all verbs regardless of their conjugation class.
This can be seen by (\ref{ex:nconj-prog}) which gives four progressive forms of verbs that belong to each of the four conjugation classes.

\pex\label{ex:nconj-prog}%
\a\label{ex:nconj-prog-0}%
\exrtcmt{∅-conj.\ progressive}%
\begingl
	\gla	Yaa @ \rlap{\Qf{a}\Af{n}\Ef{a}kélʼ.} @ {} @ {} @ {} //
	\glb	ÿaa= \Qf{a}- \Af{n}- \rt[²]{kelʼ} -μH //
	\glc	along= \Qf{\xx{arg}}- \Af{\xx{ncnj}}- \rt[²]{undo} -\xx{var} //
	\gld	along \rlap{3>3.\xx{zcnj}.\xx{prog}.undo} {} {} {} {} //
	\glft	‘S/he/it is undoing him/her/it.’
		//
\endgl
\a\label{ex:nconj-prog-n}%
\exrtcmt{n-conj.\ progressive}%
\begingl
	\gla	Yaa @ \rlap{\Qf{a}\Af{n}\Ef{a}sʼélʼ.} @ {} @ {} @ {} //
	\glb	ÿaa= \Qf{a}- \Af{n}- \rt[²]{sʼelʼ} -μH //
	\glc	along= \Qf{\xx{arg}}- \Af{\xx{ncnj}}- \rt[²]{tear} -\xx{var} //
	\gld	along \rlap{3>3.\xx{ncnj}.\xx{prog}.tear} {} {} {} {} //
	\glft	‘S/he/it is tearing him/her/it.’
		//
\endgl

In contrast, forms with \fm{\Af{\xx{cnj}}-} = \fm{\Af{n}-} have the \fm{\Af{n}-} prefix only in verbs that belong to the \fm{n}-conjugation class.
This is shown in (\ref{ex:nconj-imp}) using the same four verbs with imperative mood forms.
Only the \fm{n}-conjugation class verb \fm{aawasʼéil} (\fm{n}; achievement) ‘s/he/it tore him/her/it’ in (\ref{ex:nconj-imp-n}) appears with the \fm{\Af{n}-} prefix.

There are many contexts where a conjugation prefix \fm{\Af{\xx{cnj}}-} is specified.
All such contexts could specifically have \fm{\Af{\xx{cnj}}-} = \fm{\Af{n}-}.
Some of these contexts also include the irrealis \fm{\Rf{u}-} and/or modal \fm{\Mf{g̱}-} prefixes; these are detailed in subsections \ref{sec:nconj-irrealis} through \ref{sec:nconj-irrealis+modal}.
The seven remaining contexts listed here have \fm{\Af{\xx{cnj}}-} alone without \fm{\Rf{u}-} or \fm{\Mf{g̱}-}:

\begin{itemize}[leftmargin=0.75em]\raggedyright
\item	imperative mood \fm{\Af{\xx{cnj}}-}
\item	two specialized imperfective aspect types:
	\begin{itemize}
	\item	extensional state imperfective \fm{\Af{\xx{cnj}}-\If{i}-…-μμH}
	\item	multipositional state imperfective \fm{\Af{\xx{cnj}}-\If{i}-…-k(\:wt\:)}
	\end{itemize}
\item	habitual aspect \fm{\Af{\xx{cnj}}-…-ch} (not \fm{∅}-conj.; see sec.\ \ref{sec:pfv})
\item	realizational aspect \fm{\Af{\xx{cnj}}-\If{i}-…-μμH}
\item	consecutive aspect \fm{\Af{\xx{cnj}}-…-μμH}
\item	conditional mood \fm{\Af{\xx{cnj}}-…-n-í}
\end{itemize}

Of these seven contexts, the extenstional state imperfective, the multipositional state imperfective, and the relatizational aspect forms have the \fm{\If{i}-} stative prefix.
The two state imperfectives are lexically specified and the realizational aspect is very rare.
All three are illustrated in (\ref{ex:nconj-i}), with the realizational using the same \fm{n}-conjugation verb as in (\ref{ex:nconj-imp}).

\pex\label{ex:nconj-i}%
\a\label{ex:nconj-i-extstate}%
\exrtcmt{n-conj.\ extensional state impfv.}%
\begingl
	\gla	\rlap{\Af{N}\Ef{a}\If{a}léi.} @ {} @ {} @ {} //
	\glb	\Af{n}- \If{i}- \rt[¹]{leͥ} -μμH //
	\glc	\Af{\xx{ncnj}}- \If{\xx{stv}}- \rt[²]{far} -\xx{var} //
	\gld	\rlap{3.\xx{ncnj}.\xx{impfv}.be.far} {} {} {} //
	\glft	‘He/she/it is far, distant.’
		//
\endgl
\a\label{ex:nconj-i-mposstate}%
\exrtcmt{n-conj.\ multipositional state impfv.}%
\begingl
	\gla	\rlap{Áx̱} @ {} \rlap{\Af{n}\Ef{a}\If{a}dákwt.} @ {} @ {} @ {} @ {} @ {} //
	\glb	á -x̱ \Af{n}- \If{i}- \rt[¹]{da} -μH -kw -t //
	\glc	\xx{3n} -\xx{pert} \Af{\xx{ncnj}}- \Rf{\xx{stv}}- \rt[¹]{flow} -\xx{var} -\xx{rep} -\xx{rep} //
	\gld	it -at \rlap{3.\xx{ncnj}.\xx{impfv}.be.flow.\xx{mpos}} {} {} {} {} {} //
	\glft	‘They (waters) lie along it.’
		//
\endgl
\a\label{ex:nconj-i-realizational}%
\exrtcmt{n-conj.\ realizational}%
\begingl
	\gla	\rlap{\Qf{A}\Af{n}\Ef{a}\If{a}sʼéilʼ.} @ {} @ {} @ {} @ {} //
	\glb	\Qf{a}- \Af{n}- \If{i}- \rt[²]{sʼelʼ} -μμH //
	\glc	\Qf{\xx{arg}}- \Af{\xx{ncnj}}- \If{\xx{stv}}- \rt[²]{tear} -\xx{var} //
	\gld	\rlap{3>3.\xx{ncnj}.\xx{rlzn}.tear} {} {} {} {} //
	\glft	‘S/he/it finally tore him/her/it.’
		//
\endgl
\xe

The imperative, habitual, consecutive, and conditional lack the \fm{\If{i}-} prefix.
All of these are relatively common and are generally not limited to particular verbs.
The verb of the imperative in (\ref{ex:nconj-prog-n}) is shown with these three other forms in (\ref{ex:nconj-noi}).

\pex\label{ex:nconj-noi}%
\a\label{ex:nconj-noi-habitual}%
\exrtcmt{n-conj.\ habitual}%
\begingl
	\gla	\rlap{\Qf{A}\Af{n}\Ef{a}sʼélʼch.} @ {} @ {} @ {} @ {} //
	\glb	\Qf{a}- \Af{n}- \rt[²]{sʼelʼ} -μH -ch //
	\glc	\Qf{\xx{arg}}- \Af{\xx{ncnj}}- \rt[²]{tear} -\xx{var} -\xx{rep} //
	\gld	\rlap{3>3.\xx{ncnj}.\xx{hab}.tear} {} {} {} {} //
	\glft	‘S/he/it always tore him/her/it.’
		//
\endgl
\a\label{ex:nconj-noi-consecutive}%
\exrtcmt{n-conj.\ consecutive}%
\begingl
	\gla	\rlap{\Qf{A}\Af{n}\Ef{a}sʼéilʼ, …} @ {} @ {} @ {} @ {} //
	\glb	\Qf{a}- \Af{n}- \rt[²]{sʼelʼ} -μμH {} //
	\glc	\Qf{\xx{arg}}- \Af{\xx{ncnj}}- \rt[²]{tear} -\xx{var} \xx{sub} //
	\gld	\rlap{3>3.\xx{ncnj}.\xx{hab}.tear} {} {} {} {} //
	\glft	‘Her/his/its having finally torn him/her/it, …’
		//
\endgl
\a\label{ex:nconj-noi-conditional}%
\exrtcmt{n-conj.\ conditional}%
\begingl
	\gla	\rlap{\Qf{A}\Af{n}\Ef{a}sʼélʼni, …} @ {} @ {} @ {} //
	\glb	\Qf{a}- \Af{n}- \rt[²]{sʼelʼ} -μH -n -í //
	\glc	\Qf{\xx{arg}}- \Af{\xx{ncnj}}- \rt[²]{tear} -\xx{var} -\xx{nsfx} -\xx{sub} //
	\gld	\rlap{3>3.\xx{ncnj}.\xx{hab}.tear} {} {} {} //
	\glft	‘If s/he/it tears him/her/it, …’
		//
\endgl
\xe

All combinations of the \fm{\Df{d}-} and \fm{\Ff{s}-}/\fm{\Ff{l}-}/\fm{\Ff{sh}-} prefixes are grammatically plausible in this section, but it has yet to be determined if they all actually occur.
\end{multicols}
